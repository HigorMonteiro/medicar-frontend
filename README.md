## Quickstart

```sh
git clone git@github.com:HigorMonteiro/medicar-frontend.git
cd medicar-frontend
npm install
ng server -o
```

Note: in the path `medicar-frontend/src/app/services/user.service.ts` you can change your api host.

### Example:

```
private BASE_URL = "http://localhost:8000/api/";
```
